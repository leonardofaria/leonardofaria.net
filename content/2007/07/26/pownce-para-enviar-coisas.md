O [Pownce](http://pownce.com) é mais uma rede social com nome estranho, mas com um diferencial: [Kevin Rose](http://pownce.com/kevin) – para quem não sabe, o homem por trás do [Digg](http://www.digg.com).  
Essas &#8216;coisas' que você envia para o site podem ser arquivos, links, músicas e fotos. Criei [minha conta](http://pownce.com/leonardofaria/) no serviço e estou achando bem bacana. Com uma interface bem clean – como todas as outras aplicações web 2.0 – o Pownce permite envio de arquivos de até 100MB e possui um cliente para desktop para facilitar a interação com o serviço.

Nota: para utilizar o serviço, é necessário convite. Ainda tenho seis :)
