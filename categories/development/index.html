Você conhece o WebObjects?

WebObjects é um conjunto de servidor web Java com uma série de frameworks proprietárias, criado pela NeXT e absorvida pela Apple. Talvez seja a primeira framework para web applications, já com 11 anos. Apesar de não ter grandes cases externos, o WebObjects é chave de importantes negócios e possui clientes de peso como a BBC e o Deutsch Bank. No Brasil, a Americanas.com utiliza o framework, que já custou 50 mil dólares e hoje é distribuido gratuitamente com o xCode, o conjunto de ferramentas de desenvolvimento da Apple.
